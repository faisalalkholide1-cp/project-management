#  Project Management 

A full-stack project management  built using **Angular (Frontend)** and **PHP + MySQL (Backend)**.  
The system allows managing projects and their related tasks with full CRUD operations.

---

#  Features

##  Projects
- Create new projects
- Edit existing projects
- Delete projects
- View all projects

##  Tasks
- Create tasks under a specific project
- Delete tasks
- Toggle task status (Active / Completed)
- Filter tasks (All / Active / Completed)
- View tasks by project

---

#  System Architecture

## Frontend
- Angular (Standalone Components)
- Signals for state management
- Angular Reactive Forms
- Angular Router
- Angular Material (Dialog, Snackbar)

## Backend
- PHP REST API
- MySQL database
- Prepared Statements for security
- JSON responses
